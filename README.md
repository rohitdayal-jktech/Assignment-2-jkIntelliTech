# ✅ ToDo List API - FastAPI + MySQL

This is a simple **ToDo List REST API** built using **FastAPI**, **MySQL**, and **SQLAlchemy**, with data validation via **Pydantic**.

---

## 📁 Folder Structure

.
├── app/
│ ├── init.py
│ ├── database.py
│ ├── models.py
│ ├── schemas.py
│ └── crud.py
├── main.py
├── requirements.txt
├── .gitignore
└── README.md


---

## 🚀 Features

- Create, Read, Update, Delete ToDo items
- MySQL backend
- SQLAlchemy ORM
- Pydantic models
- Modular file structure

---

## 🛠️ Tech Stack

- Python 3.10+
- FastAPI
- MySQL (via PyMySQL)
- SQLAlchemy
- Pydantic
- Uvicorn
